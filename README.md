# FileExplorer
A data-structure-focused academic project demonstrating the implementation of BST, Doubly Linked List, and Stack in a simulated file directory system.

Console-based File Directory System in C++.

Used Binary Search Tree (BST) for directory management.

Implemented Doubly Linked List for file storage.

Developed a custom Stack to maintain recently opened files.

Supports Create, Search, and Delete operations for directories and files.
